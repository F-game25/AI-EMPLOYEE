STATE MANAGEMENT:
═════════════════════════════════════════════════════════════════════════════

CodexAnalyzer State:
  - analysis: object (null | Codex response)
  - loading: boolean
  - error: string | null
  - selectedIssue: object | null
  - severityFilter: string | null ('critical'|'high'|'medium'|'low'|null)
  - typeFilter: string | null ('bug'|'style'|'perf'|'refactoring'|null)
  - highlightedLine: number | null

Derived State (memoized):
  - allIssues: array (combined from all categories)
  - filteredIssues: array (after applying all filters)

PROPS:
═════════════════════════════════════════════════════════════════════════════

CodexAnalyzer:
  - fileId: string (unique file identifier)
  - fileName: string (file name, for language detection)
  - fileContent: string (file content to analyze)
  - onClose?: () => void (callback when user closes)

AnalysisSummary:
  - analysis: object (Codex response with bugs, style_issues, etc.)
  - issues: array (combined all issues)

CodePreview:
  - content: string (file content)
  - language?: string (programming language)
  - issues?: array (issues to highlight)
  - onLineClick?: (lineNum: number) => void
  - selectedLine?: number

IssuesList:
  - issues: array (issues to display)
  - selected?: object (currently selected issue)
  - onSelect: (issue: object) => void

IssueDetail:
  - issue: object (issue to display)
  - onClose: () => void (close callback)
